This project is a professional landing page for a real business.

## Brand
Barras Nómades

## Goal
Create a visually strong, modern, premium website for events bartending services.

## Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Priorities
- Fast delivery
- Strong visual impact
- Clean structure
- Reusable components

## UI Rules
- Premium look
- Modern and clean
- Strong hero section
- Clear CTA (WhatsApp)
- Good spacing and typography

## Architecture
- Use App Router
- Components split into sections and UI
- Avoid overengineering
- Server components by default

## Behavior
- Provide implementation-ready code
- Prioritize speed + quality
- Avoid generic layouts
